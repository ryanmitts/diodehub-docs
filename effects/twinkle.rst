*************
Twinkle
*************

Twinkles one random light on with a the chosen color.

.. Note:: Only the first color of the list is used.

.. code-block:: json
    :caption: With options
    :linenos:

    {
        "effect": "randomTwinkle",
        "options": {
            "speed": 100
        },
        "colors": [
            "#ff0000"
        ]
    }
