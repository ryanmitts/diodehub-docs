*************
Running Color
*************

Displays a sine wave type effect across the lights of the chosen color.

.. Note:: Only the first color of the list is used.

.. code-block:: json
    :caption: With options
    :linenos:

    {
        "effect": "runningColor",
        "options": {
            "speed": 10
        },
        "colors": [
            "#ff0000"
        ]
    }
