*************
Pulse In Out
*************

Pulses one color after another.

.. Note:: The speed option respresents the time each color is shown.

.. code-block:: json
    :caption: With three colors
    :linenos:

    {
        "effect": "pulseInOut",
        "options": {
            "speed": 2000
        },
        "colors": [
            "#ff0000",
            "#00ff00",
            "#0000ff"
        ]
    }
