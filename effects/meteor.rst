********
Meteor
********

Displays a group of pixels with a tail that degrades as it moved across the lights.

.. code-block:: json
    :caption: Meteor effect with options
    :linenos:

    {
        "effect": "meteor",
        "options": {
            "meteorSize": 10,
            "meteorTailDecay": 64,
            "speed": 20
        }
    }
